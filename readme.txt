== Description ==

This is an offical plugin developed by Archon llc. This plugin allows you to show off the hard earned testimonials you've collected from you works. You can use the native shortcode function of wordpress to show them in a responsive manner anywhere on your wordpress site. The plugin offers multiple customization options:

1. You can choose a specific testimonial to show.
2. You can have a testimonial picked at random.
3. You can show the most recent testimonial to show.

== Installation ==

There are a few options for installing and setting up this plugin.

= Upload Manually =

1. Download and unzip the plugin
2. Upload the 'testimonials-by-archon' folder into the '/wp-content/plugins/' directory
3. Go to the Plugins admin page and activate the plugin

= Install Via Admin Area = 

1. In the admin area go to Plugins > Add New and search for "Official Archon Testimonials Plugin"
2. Click install and then click activate

= To Setup the Plugin =
1. In the WordPress admin area go to Settings > Testimonials by Archon
2. Add new testimonial at the top of the page and click save

= How to Use the Shortcode =
1. Navigate to the post or page you would like to add the Testimonial to
2. Enter in the shortcode [testimonial]
3. Customize the testimonial that shows by adding the parameter 'id' with the id of the testimonial will show
4. Examples are [testimonial] will show the newest, [testimonial id=1] will show the testimonial with id of 1, [testimonial id=random] will show a testimonial picked at random


== Frequently Asked Questions ==

= Can I make a change to a testimonial I've already entered? =

Yes! Right under the add new testimonial form on the options page you can select a testimonial from the dropdown then edit the information you want to change and then click update. The testimonial will then be updated without changing the id of the testimonial.

= How do I know what id my testimonials have? =

On the options page under the edit testimonials dropdown the id of each testimonial is to the right of the authors name.
